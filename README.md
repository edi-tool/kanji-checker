# 常用外漢字判定ツール

入力された漢字が、常用漢字かを判定して出力するツールです。

https://edi-tool.github.io/kanji-checker/

## 概要
文化庁の常用漢字表に基づき、漢字の習得学年を特定します。
執筆や、編集工程での使用を想定しています。
